# Sort-files-in-your-directory

Steps:
  1. Locate your files by browsing. Select a directory.
  2. Choose the way of sorting:
    Sort by Name
    Sort by Date Modified
    Sort by File Type
    Sort by File Size
  3. Click "Sort"
